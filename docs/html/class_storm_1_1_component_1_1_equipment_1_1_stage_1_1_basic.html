Storm::Component::Equipment::Stage::Basic Class Reference

This partially concrete class implements the Storm::Component::Equipment::Stage interface for a single stage, or multiple stages where transitions between stages have no restrictions other than the minimum cycle on/off times. More...

Detailed Description

This partially concrete class implements the Storm::Component::Equipment::Stage interface for a single stage, or multiple stages where transitions between stages have no restrictions other than the minimum cycle on/off times.
