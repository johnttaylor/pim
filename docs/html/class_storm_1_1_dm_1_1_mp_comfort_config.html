Detailed Description

This class provides a concrete implementation for a Point who's data is the the Comfort configuration attributes.

The toJSON() method is a read/modify operation, i.e. omitted key/value fields in the 'val' object are NOT updated.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", invalid:nn, seqnum:nnnn, locked:true|false, val:{ "cmpCooling":{"stage":<num>, "cph":<enum>, "minOn":<secs> "minOff":<secs>},
"cmpHeating":{"stage":<num>, "cph":<enum>, "minOn":<secs> "minOff":<secs>},
"indoorHeating":{"stage":<num>, "cph":<enum>, "minOn":<secs> "minOff":<secs>}}}

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
