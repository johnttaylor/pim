Detailed Description

This abstract class define 'equipments' interface to a "stage".

The "stage" is responsible for determine what HVAC outputs are on based on the current loading.

◆ requestModeToOff()

virtual void Storm::Component::Equipment::StageApi::requestModeToOff ( )
pure virtualnoexcept

This method is used to 'reset' the StageApi's internal FSM to the off state.

The intended use for this method is turn off the stage when system transitions to the off operating mode and/or to an operating mode that is not appropriate for the stage's operating mode.

NOTE: For this scenario - the new operating mode is expected/required to ensure the HVAC outputs are in the proper state (i.e. clean-up previous the mode's outputs) - so that stage being requested to OFF does not set/clear any HVAC outputs.

Implemented in Storm::Component::Equipment::Stage::Basic.
