Detailed Description

This class provides a concrete implementation for Model Point with a type of Storm::Type::EquipmentTimes_T.

The toJSON() method is a read/modify operation, i.e. omitted key/value fields in the 'val' object are NOT updated.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false, val:{"beginIndoorOnTimeSec":mm.nn, "beginIndoorOffTimeSec":mm.nn, "beginOutdoorOnTimeSec":mm.nn, "beginOutdoorOffTimeSec":mm.nn} }

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
