Storm::Component::OperatingMode Class Reference

This concrete class determine the actual mode of operation for the thermostat when the user has selected "Auto Mode". More...

Detailed Description

This concrete class determine the actual mode of operation for the thermostat when the user has selected "Auto Mode".

It is also responsible for the following:

1) Determine the 'active' set-point and calculation the delta error value for use by the PI Component. 2) Populate the SystemConfig model point based on the actual operation mode.
