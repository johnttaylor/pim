Detailed Description

This class provides a concrete implementation for the virtual outputs for the system.

The virtual outputs are later mapped to relay/discrete outputs and/or digital communications to control the physical equipment.

The toJSON() method is a read/modify operation, i.e. omitted key/value fields in the 'val' object are NOT updated.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false, val:{ idFan:<out>, odFan:<out>, sovHeat:true:false, idStages:[ { stage:<num>, capacity:<out> }, ...], odStages:[ { stage:<num>, capacity:<out> }, ...]}}
where 'out' is value between 0 and 1000 that describes speed/capacity of the output. 0=off, 500 = 50% speed/capacity, 1000=on/max-capacity

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
