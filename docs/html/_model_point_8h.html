#define OPTION_CPL_DM_MODEL_POINT_STATE_INVALID   1

This symbol provides the default 'Invalid' state value for a Model Point.

The application is free define/apply its own meaning to the set of 'invalid-values'. NOTE: All 'Invalid' values MUST be greater than zero, i.e. between 1 and 127. Zero and negative values ARE Reserved by the RTE Engine.
