Cpl::Io::Serial::RP2040::Uart::InputOutput Class Reference

This concrete class implements a Input stream using Raspberry Pi UART interface/hardware. More...

Detailed Description

This concrete class implements a Input stream using Raspberry Pi UART interface/hardware.

This implement provide a more robust stream interface than using 'stdio_uart' because it is interrupted driven and provides software TX/RX FIFOs

Notes:

  1. The UART Interrupt service handlers execute on the core that executing start() method.
  2. The implement does NOT support the blocking semantics of the Cpl::Io streams. The design decision was for the implementation to be compatible/useful on bare-metal systems, i.e. no threads required.
