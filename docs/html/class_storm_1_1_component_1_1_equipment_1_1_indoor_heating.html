Storm::Component::Equipment::IndoorHeating Class Reference

This concrete class provides the business logic for controlling the HVAC system when the system is in the IndoorHeating Mode. More...

Detailed Description

This concrete class provides the business logic for controlling the HVAC system when the system is in the IndoorHeating Mode.

Specifically this class contain the logic how to use/manage up to 3 stages of indoor heating capacity

#include <IndoorHeating.h>
