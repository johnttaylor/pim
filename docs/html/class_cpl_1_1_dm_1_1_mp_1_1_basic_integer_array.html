Detailed Description

template<class ELEMTYPE>
class Cpl::Dm::Mp::BasicIntegerArray< ELEMTYPE >

This template class extends the implementation of BasicArray<ELEMTYPE> to support the toJSON() and fromJSON_() methods for integer element types.

NOTES: 1) All methods in this class are NOT thread Safe unless explicitly documented otherwise.
