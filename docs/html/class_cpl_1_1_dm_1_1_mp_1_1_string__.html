Detailed Description

template<int S, class MPTYPE>
class Cpl::Dm::Mp::String_< S, MPTYPE >

This mostly concrete template class provides the storage for a Point who's data is a null terminated string.

The template parameter specifies the size of string storage. The child classes must provide the following: 

getTypeAsText() method and a typedef for child specific 'Observer'

Template Args: S:= Max Size of the String WITHOUT the null terminator! MPTYPE:= Class type of the Child class
