60/** This class provides a concrete implementation for a Point who's data is the
61 the house/room/zone cooling and heating set-points. The class enforces
62 the min/max ranges for the set-points. Minimum delta between the two
63 set-points is NOT maintained (i.e. that is is policy decision)
64
65 The toJSON() method is a read/modify operation, i.e. omitted key/value
66 fields in the 'val' object are NOT updated.
67
68 The toJSON()/fromJSON format is:
69 \code
70
71 { name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false, val:{"cool":mm.m, "heat":nn.n} }
72
73 \endcode
74
75
76 NOTE: All methods in this class ARE thread Safe unless explicitly
77 documented otherwise.
78 */

154 {
155 m_modelDatabase.lock_();

157 float finalHeatSetpt;
