Detailed Description

template<class CONTEXT, class DATA>
class Cpl::Dm::RmwComposer< CONTEXT, DATA >

This template class is a composer pattern/class that manages the callback function for a Model Point's read-modify-write operation.


A Composer is a structural pattern that may be used to employ composition when implementing an interface rather than using multiple inheritance. This allows a single concrete object to receive the RMW callbacks from many Model Points.

Template Arguments: CONTEXT - The class that implements the Callback function DATA - The type of the Model Point Data instance.
