IntegerExpressionParser.h File Reference

Detailed Description

This file contains a collection of methods that parses a null terminated string that contains an arithmetic expression, and evaluates the expression.

This module is adapted from Kim Walisch's code at: https://github.com/kimwalisch/calculator
