Detailed Description

This concrete class is a mix of FString and a DString - the initial memory is dynamically allocated - but no additional memory allocation will occur after the instance is created, i.e.

it is a 'non-template' version - so to speak - of an FString.

NOTE: See base class - String - for a complete listing/description of the class's methods.
