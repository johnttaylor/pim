34/** This class provides a concrete implementation for the virtual outputs for
35 the system. The virtual outputs are later mapped to relay/discrete
36 outputs and/or digital communications to control the physical equipment.
37
38 The toJSON() method is a read/modify operation, i.e. omitted key/value
39 fields in the 'val' object are NOT updated.
40
41 The toJSON()/fromJSON format is:
42 \code
43
44 { name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false, val:{ idFan:<out>, odFan:<out>, sovHeat:true:false, idStages:[ { stage:<num>, capacity:<out> }, ...], odStages:[ { stage:<num>, capacity:<out> }, ...]}}
45
46 where 'out' is value between 0 and 1000 that describes speed/capacity of the output. 0=off, 500 = 50% speed/capacity, 1000=on/max-capacity
47
48 \endcode
49
50
51 NOTE: All methods in this class ARE thread Safe unless explicitly
52 documented otherwise.
53 */
