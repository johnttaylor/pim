template<class ITEM >
ITEM Cpl::Container::Stack< ITEM >::peekTop ( bool *  statusPtr = 0) const
inlinenoexcept

Returns the item on the top of the Stack.

The returned item remains on the stack. If the stack is empty, then a value of zero is returned. The optional argument 'statusPtr'is used to return stats/result of the pop operation, i.e. when *statusPtr == false then the peek operation failed and the returned value is invalid.

Removes the top item of the stack.
