LogicalButtons.h File Reference

Detailed Description

This file provide the logic for sampling the low level button drivers and generating logical buttons and button events (e.g repeat events for button pressed for 'long time).

NOTE: The initial 'pressed event' is generated on the physical RELEASE of the button OR the press-n-hold time criteria has been met.

The button events for the logical buttons are reported via Model points. The MPs will generate change notifications where their change state. Note: A repeat-event will generate change notifications.

The Model Points are: mp_buttonEventMode // aka A mp_buttonEventFan // aka B mp_buttonEventUp // aka X mp_buttonEventDown // aka Y
