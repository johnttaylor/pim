13/** @file

15 This file provide the logic for sampling the low level button drivers and
16 generating logical buttons and button events (e.g repeat events for button
17 pressed for 'long time).
18
19 NOTE: The initial 'pressed event' is generated on the physical RELEASE of
20 the button OR the press-n-hold time criteria has been met.

23 will generate change notifications where their change state. Note: A repeat-event
