Detailed Description

This class provides a concrete implementation for a Point who's data is the the Equipment Configuration attributes.

The toJSON() method is a read/modify operation, i.e. omitted key/value fields in the 'val' object are NOT updated.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", valide:true|false, seqnum:nnnn, locked:true|false, val:{"type":"<enum>", "vspMotor":true|false, "numHeat":n} }

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
