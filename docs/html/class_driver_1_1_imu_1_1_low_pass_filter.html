Detailed Description

template<class DATA_T, class FILTERED_T>
class Driver::Imu::LowPassFilter< DATA_T, FILTERED_T >

This concrete template class provides a simple Low Pass filter for a single data stream (and/or single Axis of IMU data).

See https://en.wikipedia.org/wiki/Low-pass_filter

The template arguments: DATA_T data type of the data being filtered FILTERED_T data type of filtered data (i.e the output of filter). It is also the data type use for all internal calculations.
