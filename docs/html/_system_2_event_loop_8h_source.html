33 /** This concrete class is a Runnable object that provides a event driven
34  execution model for a thread. The thread will remaining blocked until
35  an "event" occurs. The following "events" are supported:
36 
37  1) The EventLoop Semaphore was signaled
38  2) A Event Flag was set( Cpl::System::EventFlag)
39  3) A Software Timer expiring (Cpl::System::Timer)
40 
41  To receive a callbacks when Event Flags are set, the application must create
42  a child class and provide its own implementation of the processEventFlag()
43  method.
44 
45  The class also provides a mechanism for registering 'other' events to
46  be processed.
47 
48  Note: The EventLoop does NOT use/consume the Thread Semaphore.
49  */
50 class EventLoop : public Runnable, public EventFlag, public Signable, public TimerManager
51 {
52 public:
53  /** Constructor. The 'timeOutPeriodInMsec' parameter specifies how
54  long the EventLoop will wait for an event before timing out and
55  processing the software timers. The value of 'timeOutPeriodInMsec' is
56  the resolution of the timing source for the Software Timers, i.e. the
57  event loop will 'wake-up' at least every 'timeOutPeriodInMsec'
58  milliseconds.
59 
60  A fatal error is generated if 'timeOutPeriodInMsec' is set to zero.
61  */
