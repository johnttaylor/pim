Detailed Description

template<class T>
class Driver::Imu::Quat< T >

This template class contains the quaternion results from an IMU.

For methods that use array type of notation the following mapping applies: [0] = W [1] = X-Axis [2] = Y-Axis [3] = Z-Axis
