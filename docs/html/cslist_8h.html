cslist.h File Reference

Detailed Description

This file defines a C-based interfaces for Singly lists that do not use or require dynamic memory allocation.

See the 'README.txt' file for details on how intrusive containers work.

NOTE: The list operations are NOT type safe - it the Application's responsibility for managing the types/content of the lists

◆ Cpl_Container_SList_next()

void * Cpl_Container_SList_next ( const void *  item)

This function returns the next item in the list AFTER the specified item.

The returned item is NOT removed from the list. This method is typically use to traverse the list without changing its list membership. If there is no next item the method returns 0. The method will also return 0 if the specified item is NOT in the list.
