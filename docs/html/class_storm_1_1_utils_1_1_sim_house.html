Detailed Description

This class provides a 'functional' simulation of house with respect to Outdoor temperature, Indoor temperature, and active HVAC capacity.

Essentially, this class provide functional feedback for a thermostat's control algorithm. The feedback is NOT representative of actual house/thermal dynamics, but is sufficient to functional test a control algorithm. This class uses the SimSystem class as basis for the feedback.

Some terminology, notes, design, etc.

maxOdt = Maximum Outdoor temperature for the simulation in degrees Fahrenheit minOdt = Minimum Outdoor temperature for the simulation in degrees Fahrenheit SimSystem.maxPotenial = maxOdt - minOdt; coolingCapacity = -(SimSystem.maxPotenial * odtCoolingLoadRating), e.g. -(140' * 0.33) = -46.2' heatingCapacity = SimSystem.maxPotenial * odtHeatingLoadRating, e.g. 140' *1.0 = 140' percentActivityCapacity = Current active HVAC capacity. 0.0 = No active capacity, 1.0 = Full Capacity inPotOdt = current outdoor temperature - minOdt InPotHVAC = (coolingCapacity? coolingCapacity : heatingCapacity) * percentActivityCapacity SimSystem.InPot = inPotOdt + InPotHVAC Indoor Temperature = SimSystem.OutPot + minOdt
