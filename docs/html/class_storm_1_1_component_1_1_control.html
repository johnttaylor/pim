Storm::Component::Control Class Reference

This concrete class provides the Component wrapper that is responsible for determining how to control the HVAC equipment to meet the current load and/or user inputs. More...

Detailed Description

This concrete class provides the Component wrapper that is responsible for determining how to control the HVAC equipment to meet the current load and/or user inputs.

The concrete Equipment object (provide as a constructor argument) implements the actual control business logic.
