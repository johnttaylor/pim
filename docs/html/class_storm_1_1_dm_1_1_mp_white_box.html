Detailed Description

This class provides a concrete implementation for a Point who's data is a collection of variables used to facilitate white box testing.

The toJSON() method is a read/modify operation, i.e. omitted key/value fields in the 'val' object are NOT updated.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false, val:{"defeatEquipMinOffTime":true|false, "abortOnOffCycle":true|false} }

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
