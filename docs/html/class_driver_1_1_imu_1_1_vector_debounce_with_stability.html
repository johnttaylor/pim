Detailed Description

template<class T>
class Driver::Imu::VectorDebounceWithStability< T >

This concrete class provides a basic debounce filter algorithm across all axises of the Vector.

In addition it supports the concept of stability, i.e. the debounced value has not changed for at least N samples.

#include <VectorDebounceWithStability.h>
