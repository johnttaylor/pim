◆ executeActive()

virtual bool Storm::Component::Control::Equipment::executeActive ( Args_T & args)
pure virtualnoexcept

This method will be called on a periodic basis (as determined by the calling Control Component instance) to perform active conditioning.

This method will only be called when the current operating mode 'matches' the operating configuration of the Equipment instance.

The method returns true if no error(s) occurred; else false is returned. Note: If false is return, then the Component's MP HVAC outputs will be updated to an "all off state" (the Outdoor SOV output will not be changed if/when this happens).

Implemented in Storm::Component::Equipment::Cooling, Storm::Component::Equipment::IndoorHeating, and Storm::Component::Equipment::Off.

◆ executeOff()

virtual bool Storm::Component::Control::Equipment::executeOff ( Args_T & args)
pure virtualnoexcept

This method will be called on a periodic basis (as determined by the calling Control Component instance) to perform active conditioning.

This method will only be called when the current operating mode DOES NOT MATCH the operating configuration of the Equipment instance.

The method returns true if no error(s) occurred; else false is returned.

Note: This method NEVER updates the Component's MP HVAC outputs.

Implemented in Storm::Component::Equipment::Cooling, Storm::Component::Equipment::IndoorHeating, and Storm::Component::Equipment::Off.
