#define Cpl_Io_Serial_Renesas_Rx62n_Hal_uartInit   Cpl_Io_Serial_Renesas_Rx62n_Hal_uartInit_MAP

This method provides an initialization routine for the micro-controller's UARTs.

Parameters
sciPortIDUART/SCI Port ID (must be one of BSP_ID_SERIAL_PORTx)
pinSelectPin select, i.e. Pin configuration 'A' vs 'B'. Not all pin configuration options are available on all packages of the microcontroller
baudrateBRR value
baudrateDividerPCLK divider/prescaler. 0= PCLK, 1=PCLK/4, 2=PCLK/16, 3=PCLK/11
irqPriorityInterrupt Priority. 1=lowest, ... 15=highest
frameConfigis set by bit-wise ORing the Parity, Stopbits, and data length values together
Returns
1 if successful; else 0 is returned. Note: The error checking is VERY LIMITED, i.e. it only checks if 'sciPortID' is a valid ID.
