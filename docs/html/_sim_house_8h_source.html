23/** This class provides a 'functional' simulation of house with respect to
24 Outdoor temperature, Indoor temperature, and active HVAC capacity. Essentially,
25 this class provide functional feedback for a thermostat's control algorithm.
26 The feedback is NOT representative of actual house/thermal dynamics, but is
27 sufficient to functional test a control algorithm. This class uses the
28 SimSystem class as basis for the feedback.
29
30 Some terminology, notes, design, etc.
31
32 maxOdt = Maximum Outdoor temperature for the simulation in degrees Fahrenheit
33 minOdt = Minimum Outdoor temperature for the simulation in degrees Fahrenheit
34 SimSystem.maxPotenial = maxOdt - minOdt;
35 coolingCapacity = -(SimSystem.maxPotenial * odtCoolingLoadRating), e.g. -(140' * 0.33) = -46.2'
36 heatingCapacity = SimSystem.maxPotenial * odtHeatingLoadRating, e.g. 140' *1.0 = 140'
37 percentActivityCapacity = Current active HVAC capacity. 0.0 = No active capacity, 1.0 = Full Capacity
38 inPotOdt = current outdoor temperature - minOdt
39 InPotHVAC = (coolingCapacity? coolingCapacity : heatingCapacity) * percentActivityCapacity
40 SimSystem.InPot = inPotOdt + InPotHVAC
41 Indoor Temperature = SimSystem.OutPot + minOdt
42*/
