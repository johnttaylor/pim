Cpl::Dm::Mp::Double Class Reference

This class provides a concrete implementation for a Point who's data is a double. More...

Detailed Description

This class provides a concrete implementation for a Point who's data is a double.

The toJSON()/fromJSON format is:

{ name:"<mpname>", type:"<mptypestring>", invalid:nn, seqnum:nnnn, locked:true|false, val:<numvalue> }

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
