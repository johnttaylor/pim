Detailed Description

This class implements an asynchronous ReturnHandler.

When the rts() method of the message associated with this class is invoked, the message referenced as a member variable of this class is posted to the mailbox referenced as a member variable of this class. It is expected that the referenced mailbox belongs to the client (i.e. the sender of the original message), and that the message contains a reference to the original message, such that the original message can be released by the client.
