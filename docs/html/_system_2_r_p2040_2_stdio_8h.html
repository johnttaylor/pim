Stdio.h File Reference

Detailed Description

This interface returns the Cpl Io Stream for the for SDK's stdio.

Note: This interface works BEST when using the SDK's stdio-over-USB because its implementation supports software FIFO for the streams (where as stdio-over-UART only support the hardware FIFOs)
