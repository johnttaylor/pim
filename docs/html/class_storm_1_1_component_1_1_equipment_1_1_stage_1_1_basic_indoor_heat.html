Detailed Description

This concrete class implements the Storm::Component::Equipment::Stage interface for a single compressor cooling stage, or multiple compressor cooling stages where transitions between stages have no restrictions that the Stage object needs-to/is-required to support (i.e.

all stage transition logic is handled by the Equipment instance).
