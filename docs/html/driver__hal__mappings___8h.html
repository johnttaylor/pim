driver_hal_mappings_.h File Reference

Detailed Description

This file provides the LCONFIG mapping of various Cpl::Driver::xxxx Hardware abstraction interfaces to a null/empty implementation.

DO NOT include this file directly! Instead include the generic BSP interface - //colony.bsp/src/Bsp/Api.h.
